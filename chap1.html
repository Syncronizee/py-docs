A Brief Introduction To Python

Python itself is a very powerful programming language, very easy to read and also used for many purposes. People use Python for web apps, data analyzation, and a lot more. Check out this article (credits to W3Schools) for more information.

Setting Up Your Work Environment

To set up your work environment, you will need a good and reliable IDE (integrated development environment). For this, I recommend using Visual Studio Code (commonly abbreviated as VS Code) or Atom.

Also you will need to set up the Command Line (Terminal for MacOS and Command Prompt for Windows). This is very important for important stuff like installing packages or pushing files to GitHub.
